Utilizand laboratorul 6, si propozitiile generate din fisierul propozitii.txt, se cer urmatoarele:
1. Sa se creeze manual un fisier propozitiiMagice.txt ce va contine pe fiecare linie cate o propozitie;
2. Sa se verifice daca in fisierul propozitii.txt exista vreo propozitie identica cu o propozitie din
propozitiiMagice.txt. In caz ca exista, se va afisa pe ecran propozitia, si numarul liniei pe care aceasta
apare in fisierul propozitii.txt.
3. Pentru fiecare dintre propozitiile existente in propozitii.txt se va determina si afisa pe ecran procentul
de echivalenta cu fiecare dintre propozitiile magice propozitiimagice.txt. Daca procentul este 0, nu se va
afisa.
Exemplu: Fisierul propozitii.txt Fisierul propozitiiMagice.txt
Marul este rosu. Casa este mare.
Telefonul suna tare. Ardeiul este rosu.
Casa arata bine.
Jocul pare greu.
Aragazul functioneaza corect.
Leopardul fuge repede. Casa este mare.
Propozitii identice:
Casa este mare - identica, linia 7
Procent de echivalenta:
Marul este rosu
- Casa este mare - 33% identica (1 cuvant din 3 (totalul) identic)
- Ardeiul este rosu - 66% identica ( 2 cuvinte din 3(totalul) identice)
Casa arata bine
- Casa este mare - 33% identica (1 cuvant din 3 (totalul) identic)
